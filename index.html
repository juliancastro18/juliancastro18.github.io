What is this?

This 3D model corresponds to the final project for the Introduction to Object Design subject when I was studying a degree in Graphic Design. The task was to create a three-dimensional object using the student's initials. An analysis of the letters had to be carried out, evaluating possible connections and interweaving between them.

It was created using the software Cinema 4D and made out of plywood. It was implemented on the web using Sketchfab's Viewer API.
